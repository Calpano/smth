## browser_list_devices

Returns all device names that can be passed as the `device` parameter to `browser_see_visual`.

Names must match exactly, e.g. `"iPhone SE"`, `"Pixel 5"`, `"iPad Mini"`. Landscape variants are separate entries: `"Pixel 5 landscape"`.

No parameters.

### Example

```
browser_list_devices
```
